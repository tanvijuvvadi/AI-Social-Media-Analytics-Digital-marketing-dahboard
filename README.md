# 🚀 AI-Powered Social Media Analytics Dashboard

## 📊 Project Overview
This project is an **AI-driven Social Media Analytics system** built using SQL for data storage and analysis. It helps track and evaluate key marketing KPIs such as engagement, impressions, reach, sentiment score, and conversion rate across multiple months.

The goal is to transform raw social media data into meaningful business insights for better marketing decision-making.

---

## 🎯 Objectives
- Analyze social media performance trends over time
- Measure engagement, impressions, and reach effectiveness
- Track sentiment score and conversion rate
- Identify best-performing months and campaigns
- Generate data-driven marketing insights

---

## 🛠️ Tech Stack
- SQL (MySQL)
- Data Analysis
- Business Intelligence Concepts
- Power BI (optional for dashboard visualization)

---

## 📂 Database Structure

### Table: `social_media_performance`

| Column Name         | Description |
|---------------------|-------------|
| performance_id      | Unique ID for each record |
| month_name          | Month of performance |
| total_engagements   | Total user engagements |
| engagement_rate     | Engagement percentage |
| impressions         | Total impressions |
| reach_count         | Total audience reach |
| sentiment_score     | Customer sentiment analysis score |
| response_time       | Average response time |
| conversion_rate     | Conversion percentage |
